Whenever you make changes and want to deploy:

```bash

```

This single command will:
1. Build the production version
2. Deploy to GitHub Pages automatically

---

## ⚙️ Manual GitHub Pages Configuration

If the site doesn't load immediately, configure GitHub Pages manually:

1. Go to: https://github.com/Rind-AI/aura-ai-premium-showcase/settings/pages
2. Under **"Source"**, select **"Deploy from a branch"**
3. Select **"gh-pages"** branch and **"/ (root)"** folder
4. Click **"Save"**
5. Wait 1-2 minutes for deployment

For local development, create `.env` file:

```env

```

**Note**: GitHub Pages is static hosting, so backend features requiring server-side API keys won't work. For full functionality, consider deploying to Vercel or similar platforms.
