# KR-NNAI: Aura AI - Premium Digital Showcase

A high-end, Figma-inspired multipage website featuring Gemini-powered intelligence, interactive chatbot, and cinematic animations.

## 🚀 Overview

KR-NNAI is the ultimate "Infinity OS" for digital showcases. It is designed to be a fully adaptable platform for brands, tech innovators, and community leaders.

### Key Features:
- **Master Admin Hub:** Real-time content editing, logo management, and site customization.
- **Gemini Data Science Tool:** Grounded AI research engine for deep industry insights.
- **Interactive AI Chatbot:** Real-time assistance powered by Gemini.
- **Dynamic Themes:** 8 premium color palettes and 5 curated font styles.
- **Cinematic Media:** Support for high-fidelity images and videos with native controls.
- **Export Ready:** Download your customized showcase as a standalone HTML product.

## 🛠️ Tech Stack
- **Frontend:** React 19, Vite, TypeScript
- **Styling:** Tailwind CSS 4
- **Animations:** Motion (formerly Framer Motion)
- **AI Engine:** Google Gemini API (@google/genai)
- **Icons:** Lucide React

## 📦 Getting Started

1. **Install Dependencies:**
   ```bash
   npm install
   ```

2. **Set Up Environment Variables:**
   Create a `.env` file and add your Gemini API Key:
   ```env
   GEMINI_API_KEY=your_api_key_here
   ```

3. **Run Development Server:**
   ```bash
   npm run dev
   ```

4. **Build for Production:**
   ```bash
   npm run build
   ```

## 🔐 Admin Access
To enter the Master Hub and edit content:
1. Click the **Lock icon** at the bottom-left.
2. Enter the Master PIN (Default: `1106`).
